# Arquitetura das propostas comerciais Studio Search

O conjunto de propostas comerciais do **Studio Search** será recriado a partir de um único sistema de apresentação, com linguagem visual consistente, estrutura modular e campos personalizáveis para o nome do cliente. A lógica será a de uma proposta consultiva premium: cada documento terá a mesma espinha dorsal, variando apenas a combinação de serviços, o texto do escopo, o cronograma e o investimento.

## Estrutura padrão para todas as propostas

| Seção | Objetivo | Conteúdo-base |
| --- | --- | --- |
| Capa | Gerar percepção de valor | Título “Proposta Comercial”, nome do cliente, assinatura Studio Search e data |
| Abertura | Personalizar a apresentação | Saudação com placeholder do cliente e texto curto de contextualização |
| Sobre o Studio Search | Reforçar autoridade | Texto institucional reescrito em tom mais estratégico e sofisticado |
| Solução Proposta | Explicar o foco da proposta | Resumo executivo com os serviços incluídos naquele documento |
| Escopo Detalhado | Especificar entregas | Módulos de entregáveis, formato de atuação e diferenciais do serviço |
| Cronograma e Fluxo | Dar previsibilidade | Prazo, etapas, aprovações e política de alterações |
| Investimento | Facilitar decisão | Valor, condição comercial e observações |
| Diferenciais | Justificar posicionamento | Identidade, imersão, direção estratégica, consistência visual e acompanhamento |
| Próximos Passos | Conduzir fechamento | Aprovação, briefing, início do projeto e contato |
| Encerramento | Fechar com marca | Assinatura Studio Search e frase final de posicionamento |

## Diretriz de redação

A redação deve ser consultiva, clara e profissional. O documento não deve soar como tabela de preços isolada, mas como uma proposta estratégica de uma boutique de marketing. Os textos devem ser mais enxutos, com melhor hierarquia e mais percepção de valor do que nos PDFs antigos.

## Diretriz de apresentação visual

A proposta deve combinar uma **capa escura premium** com páginas internas claras, baseadas em grid sutil, uso de lavanda suave e acentos em verde-neon. Essa linguagem será aplicada em todas as oito versões para garantir unidade de marca e reconhecimento imediato do Studio Search.
